4. Create `src/components/FloatingCTA.tsx` — a "use client" component rendering a `motion.a` element:
   - Fixed position: `fixed bottom-6 right-6 z-40`
   - Styled: `bg-brand-accent text-brand-text px-6 py-3 rounded-full font-medium shadow-lg`
   - href="#contact"
   - Text: "Let's talk"
   - Motion props: `whileHover={{ scale: 1.05 }}`, `whileTap={{ scale: 0.95 }}`, `initial={{ opacity: 0, y: 20 }}`, `animate={{ opacity: 1, y: 0 }}`, `transition={{ delay: 1, duration: 0.4 }}`
   - Add FloatingCTA to layout.tsx inside the Providers wrapper, after {children}, so it appears on all pages.

   - Spread remaining props onto the motion.a element
   - Export as named export `AnimatedButton`

Import from `motion/react` (NOT `framer-motion`).
